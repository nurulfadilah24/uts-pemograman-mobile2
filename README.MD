## MONEY TRACK APP

## Deskripsi
Money track adalag aplikasi mobile sederhana yang digunakan untuk mecatat dan mengelola keuangan pribadi. aplikasi ini membantu pengguna dalam mecatat pemasukan, pengeluaran, serta melihat laporan keuangan secara singkat.

## TUJUAN
- membantu pengguna mengontrol keuangan harian ataupun bulanan.
- menyediakan laporan pemasukan dan pengeluaran.
- memberikan informasi saldo secara real-time.

## FITUR UTAMA
1. Login dan Register user
2. yang belum punya akun daftar dulu.
3. input pemasukan dan pengeluaran.
4. laporan keuangan.
5. manajemen profil.
6. logout sistem.
